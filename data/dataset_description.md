# Dataset Description

**Dataset Name**: Flux Hyperlocal Delivery Data
**Source**: Internal simulated platform data (preprocessed SQLite and `.rds` files)

**Description**:
Contains comprehensive delivery, agent, and customer review data for a hyperlocal delivery platform. It includes fields for traffic level, order value, delivery times, and text reviews to enable multi-dimensional Data Mining.

**Number of instances**: ~10,000 (varies by table)  
**Number of attributes**: ~30 distinct attributes across fact and dimension tables.

**How to download**:
Data is either natively present in the `data/` directory or generated via ETL scripts. Wait for `preprocessed_data_enhanced.rds` to be generated by `1_data_preparation.R`.
